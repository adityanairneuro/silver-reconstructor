SilberReconstructor, A hub for the automated reconstruction of neuronal morphology

Aditya Nair, Gilad Silberberg

This website describes the working and use of SilberReconstructor, a hub for the automated reconstruction of neuronal morphology.

SilberReconstructor is an image processing hub developed in the language Wolfram Mathematica.
Its current applications include the automated reconstruction of neuronal morphology in third dimensional fluorescence confocal microscope images.

As SilberReconstructor uses Wolfram Mathematica to perform its computations, this software must first be installed for AdCount to be functional.

SilberReconstructor uses the popular APP2 and APP1 reconstruction algorithms (Peng et al., 2011, Xiao et al., 2013) from the Vaa3D platform (Peng et al., 2014), combining it with pre-processing algorithms written in Mathematica that specifically targets the types of noise seens in confocal images. In addition to this, SilberReconstructor also performs several post processing fucntions, thus acting as a good companion program to Vaa3D with its reconstruction algorithms
