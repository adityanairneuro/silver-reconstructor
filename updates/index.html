SilberReconstructor, A hub for the automated reconstruction of neuronal morphology

Aditya Nair, Gilad Silberberg

This website describes the working and use of SilberReconstructor, a hub for the automated reconstruction of neuronal morphology.

Current Version

SilberReconstructor 1.0

This version can perform automated reconstruction of neurons in three dimensional z stack confocal microscopy images.

SilberReconstructor 1.1 Download

This version adds batch processing of multiple neurons simultaneously. It also allows for some SWC editing such as flipping of SWC files (required to correct the change in orientation that APP2 induces)

SilberReconstructor 1.2 Download

This version adds batch processing of multiple neurons simultaneously with both intensity and sized based filtering.

SilberReconstructor 1.3 Beta Download

This version adds batch processing of multiple neurons simultaneously with both intensity and sized based filtering for upto 10 neurons.

Upcoming Versions

The next version will feature the ability to analyze morphology reconstructed by quantifying several features such neurite lenght, dendritic spine density etc

Know more about how to use SilberReconstructor using the SilberReconstructor User Manual found at this link: SilberReconstructor User Manual.
